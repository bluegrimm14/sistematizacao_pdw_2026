# Landing Page - 🏥Clínica YourHealth

Esta é uma Landing Page Simples, que visa colocar em prática conhecimentos báscios de HTL e CSS em uma página Estática.

## O que é uma Landing Page? 🤓
Landing Pages são páginas estáticas feitas com HTML e CSS que visam a apresentação de soluções e captura de possíveis clientes/leads. Esta, normalmente, é a primeira forma de contato virtual com uma marca e seus objetivos.

## Como usar? 🤔

1. Acesse ao repositório da Landing Page em https://github.com/bluegrimm14/sistematizacao_pdw_2026.git

2. Em sua IDE favorita, acesse o seu terminal e navegue para a pasta onde queira clonar o repositório:
    ```bash
    cd c:/Users/Desktop/Clinica/
    ```
3. Na pasta escolhida, clone o repositório através do seguinte comando:
    ```bash
    git clone <url_do_repositório>
    ```
- No espaço entre colchetes anguláres - tags(<>) - cole o link do repositório desejado.

4. Acesse o arquivo ``` index.html ```e abre em seu navegador preferido e *Voilá!*
